## Keyboard Shortcuts

### VSCode

- Generate basic html boilerplate: ! - TAB
- Install extension live server
- Access terminal through Control + \`
- Comment out line through Command + /
- Move block of text through Option + Up/Down

### Chrome

- Right click > Inspect > Console
- Option + Command + i
